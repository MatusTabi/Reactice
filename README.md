# Reactice

Next.js app with Auth.js + Drizzle + Turso.

## Setup

```bash
npm install
cp .env.local.example .env.local
```

Fill `.env.local`:

Push schema:

```bash
npm run db:push
```

## Run

```bash
npm run dev
```

Open: http://localhost:3000

## Useful commands

```bash
npm run build
npm run start
npm run lint
npm run db:push
npm run db:studio
```
